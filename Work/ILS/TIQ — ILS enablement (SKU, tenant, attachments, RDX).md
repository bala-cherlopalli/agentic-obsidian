# TIQ — ILS enablement (SKU, tenant, attachments, RDX)

Consolidated **product / platform gaps** called out for **enabling ILS flows in TIQ**.

## ILS client SKU

- **New SKU for ILS clients** — uses the **TIQ app**.
- **Testing**: need a way to run / validate **TIQ-only tenant** (isolated tenant for ILS-focused testing).

## Program attachments

- Need capability to **attach PDFs, spreadsheets, and documents** (reports, supporting detail, etc.) to a **program**.

## Risk Data Exchange (RDX)

- **Enhancement**: **Risk Data Exchange should support importing a program into an existing `programSet`** (not only greenfield program creation).
